# pro-leyu-lab

## 项目简介

pro-leyu-lab 是一个用于归档和发布独立 HTML 页面的仓库。这里存放的页面内容多样，涵盖工具、实验性小作品、信息展示等，不针对任何特定网站或服务。

本仓库主要服务于个人学习、测试与文档记录，所有页面均为静态 HTML 文件，可通过 GitHub Pages 直接访问。

## 目录结构

```
pro-leyu-lab/
├── index.html          # 仓库主页（可选）
├── pages/              # 独立页面归档
│   ├── page-a/         # 示例页面文件夹
│   ├── page-b/
│   └── ...
├── assets/             # 公用资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立的 HTML 页面，每个页面可包含自己的资源文件。
- `assets/`：存放可供多个页面复用的公共资源。

## 页面归档说明

- 每个独立页面通常位于 `pages/` 下的子文件夹中，并包含一个 `index.html` 作为入口。
- 页面内容可能涉及个人实验、学习笔记、工具界面等，仅供参考。
- 不提供对外链接、商业推广或诱导性内容。
- 部分页面可能包含外部资源（如 CDN 库），请确保网络通畅。
